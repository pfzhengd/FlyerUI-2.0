### Alert - 可定制属性

| 属性名称 | 类型    | 默认值  | 可选值                            | 说明                  |
| -------- | ------- | ------- | --------------------------------- | --------------------- |
| type     | String  | success | success / warning / info / danger | 原生 input 的 type 值 |
| closable | Boolean | flase   | true / false                      | 是否设置为可关闭      |
| title    | String  | -       | -                                 | 提示的标题            |

### Input - Slot

| 事件名称 | 说明                     |
| -------- | ------------------------ |
| title    | 自定义 title 内容的 slot |
